Eclipse ESCET™ contact and support

Thanks for your interest in the Eclipse Supervisory Control Engineering Toolkit (Eclipse ESCET) open-source project.

Contact the community

If you have any questions regarding the Eclipse ESCET project, any of its tools, or how to use them, feel free to ask a question on the Eclipse ESCET forum. The Eclipse ESCET developers and larger community will be happy to answer your questions.

Eclipse ESCET forum

Bugs and feature requests

If you've found a bug or have a feature request, you can create an issue in the Eclipse ESCET GitLab. Please look for existing issues before creating a new one.

For more information on how we work with issues, see our development process. Remember that contributions are always welcome!
