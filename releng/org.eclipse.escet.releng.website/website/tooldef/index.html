ToolDef

ToolDef is a cross-platform and machine-independent scripting language.

Learn more

Features

Intuitive language

ToolDef features a simple and intuitive syntax that makes it easy to write scripts.

Reduce mistakes

Static typing reduces simple mistakes, compared to Windows batch files and Linux shell scripts.

Powerful tools

ToolDef features many built-in data types and tools, and integrates well with Eclipse ESCET tools and Java.

Learn more

Getting started

The ToolDef tooling is part of the Eclipse ESCET toolkit.

It is available for Windows, Linux and macOS, portable and ready to go.
