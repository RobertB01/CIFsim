[[use-remove]]
== Removing Eclipse ESCET IDE

Before removing a version of the Eclipse ESCET IDE, you may want to preserve (back up) its <<use-terminology-workspace,workspace>> directory, to keep your data (settings, projects, files).
Then, to remove a version, simply remove the directory that contains files that you extracted from the downloaded archive.
